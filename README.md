## 📌使用说明

1. **本地查看笔记**：

   克隆仓库到本地后，直接使用Typora寻找对应文件目录，打开`.md`文件即可。

2. **图片路径说明**：

   使用Typora相对路径，存储于对应`{filename}.assets`文件夹中，请勿手动移动该文件夹。

3. **同步更新**：

   本地修改/新增笔记后，通过SourceTree完成「暂存 → 提交 → 推送」，即可同步到本仓库。
