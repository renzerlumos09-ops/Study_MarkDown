# 2026 年申请博士计划表

## 核心能力构建

本板块旨在系统性地填补我目前知识架构中地关键缺口，并将学习过程转会阿伟可视、可验证的证据。

**所有知识以 Typora 笔记记录，并定期上传至我的 GitHub：”公开课学习笔记”文件夹中**。

- 数学与统计基础：以“能用 Python 实现并绘图/输出”为标准，学习线性代数(矩阵分解)、动力系统、微积分、概率统计。

- 计算神经科学：学习 Bilibili 的神经科学网课，了解、熟悉脉冲/循环神经网络、点过程模型、降维技术、状态空间模型、原学习、多智体逆强化学习。

- 专项深度学习：每周 1 篇的频率持续阅读目标导师及领域的近 2 年核心文献，理解其数学与物理思维。

- 数据实战锻炼：寻找公开神经科学数据集(CRCNS 等)，用 Python 完整处理一次数据(尖峰序列/钙成像)，完成从预处理到降维/解码的分析流程。

本板块旨在将我学习与思考内容，固化为申请时可直接展示的证明与证据链。

- “Virtual Staining”独立项目档案：产出完整的硕士 IP 档案，包括：代码库、分析报告、清晰的图示、详细实验/分析报告、中期报告与最终答辩、导师评价等等内容，作为代码与数学能力的核心证明。

  - GitHub 代码库(文献复现代码、课程学习笔记与代码、数据分析项目、代码学习笔记)
  - 个人学术网站(GitHub Pages 搭建，集中展示项目、代码和学习轨迹)
  - 结构化的自学路径文档：自学课程列表与证书。
- RP 草案：初期形成一份 2-3 页的草案，清晰阐述科学问题、技术路径及其目标实验室的关联，并准备根据 feedback 迭代。



## 导师网络与信息

本版块旨在策略性地与目标学术圈建立联系，获取关键信息与指导并避免闭门造车。

- 系统性筛选导师：建立潜在导师池(港五校、内地研究院)，分析其研究方向、论文署名模式及学生去向。
- 寻找”学术教练“：联系目标实验室的在读博士生或年轻学者，获取内部信息和申请指导。
- 参与 NMA 等学术社群：参加计算神经科学夏利营，主动领导小组团队并与讲师、在职人员建立联系。
- 执行探索性联系：在准备好初步材料后，向最匹配的 1-2 位导师或其学生发送邮件，寻求对研究构想的非正式反馈。



## 申请流程与节点管理

本版块会将我分散的时间点集合成清晰的阶段性目标，确保流程稳步推进。

1. 基础期(1 月-2 月)：筛选导师与研究领域，鉴别是否处于招生阶段；预习下半学期课程，提前做好基础；学习计算神经科学、高等数学、神经动力学、神经流形等基础知识公开课；了解 NMA 夏利营内容，预定 NMA 夏利营席位；继续开展硕士 IP 项目，保证质量与汇报频率(与 Mentor 交流想法)等建立良好联系。
2. 准备期(3 月-6 月)：跟紧硕士课程，不落下研究生课程的 GPA；和组内学姐 、教授打好关系，为未来寻求推荐信提供基础。
3. 联络与深化期(7 月-9 月)：启动第一轮套磁，迭代 RP；参加 NMA。
4. 面试与申请期(10 月-次年 3 月)：准备面试，积极回复和教授保持联系，提交正式申请。
5. 后备方案(次年 6 月)：若无理想 offer，立刻寻求 RA 或实习岗位，积累科研经历。



## 学业基础保障

本版块会确保我的所有申请都建立在硕士学业顺利完成的坚实基础上，这是本计划完成的前提：

- GPA 管理：确保所有课程不低于 B+，优先分配精力给与未来研究强相关的课程。
- 毕业流程推荐：有限保障”独立项目”的进度与质量，顺利完成项目答辩和毕业论文、毕业答辩。
- 推荐信铺垫：通过与 IP 项目及相关课程教授的积极、专业互动，为获得有力推荐信打下基础。
